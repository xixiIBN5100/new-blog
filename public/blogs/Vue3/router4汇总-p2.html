    # 动态路由

    我们一般使用动态路由都是后台会返回一个路由表前端通过调接口拿到后处理(后端处理路由）

    主要使用的方法就是router.addRoute

    const initRouter = async () => {
    const res = await getRouter()
    res.data.route.forEach((item: any) => {
        router.addRoute({
            path: item.path,
            name: item.name,
            component: () => import(`../componts/${item.component}.vue`),//使用字符串拼接匹配组件
        })
    })
}
